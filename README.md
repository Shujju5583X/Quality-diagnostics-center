# Lab System

This is a comprehensive Medical Lab Management System built using .NET C#. 

## Architecture
- **LabSystem.Core**: Core domain models, interfaces, and shared logic.
- **LabSystem.Data**: Data access layer and database context.
- **LabSystem.Services**: Business logic and application services.
- **LabSystem.UI**: User Interface (Console or Web based).
- **LabSystem.Tests**: Unit and integration tests.

## Getting Started

To build and run the project:

```bash
dotnet restore
dotnet build
dotnet run --project LabSystem.UI
```
